## docker에서 이미지를 다운받는 법  

### docker pull  명령어를 사용한다.

태그를 붙이지 않고 이미지를 다운받으면 기본적으로 lastest 태그가 적용된다.

**lastest** : 가장 최신의 버전을 의미한다.
